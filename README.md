# SIMBA 云端 A 股模拟盘

这是一个只使用虚拟资金的 A 股云端模拟交易项目。初始资金为 100,000 元，GitHub Actions 在交易日自动获取公开行情、更新账户并执行规则化策略；GitHub Pages 展示净值、持仓、成交和最近决策。

## 云端计划

- 北京时间 09:40：更新早盘行情。
- 北京时间 13:30：更新午后行情。
- 北京时间 14:40：更新行情并执行一次每日调仓。
- 休市、行情日期过期或有效报价不足时拒绝交易。

## 模拟规则

- 买入按 100 股的整数倍。
- 当日买入次日才可卖出（简化 T+1）。
- 最多持有 4 只股票，单只目标仓位约 20%，保留约 15% 现金。
- 浮亏达到约 5% 止损，浮盈达到约 12% 止盈；当日明显走弱时退出可卖持仓。
- 佣金按万分之 2.5、最低 5 元，卖出印花税按万分之 5，过户费按十万分之 1 模拟。

## 文件

- `cloud/trade.js`：行情解析、账户估值和交易策略。
- `data/account.json`：云端账户的权威状态。
- `docs/`：公开展示页面及账户镜像。
- `.github/workflows/cloud-trade.yml`：定时交易任务。
- `.github/workflows/pages.yml`：GitHub Pages 部署。

本项目不连接券商，不会产生真实证券委托。公开行情可能存在延迟或中断，不构成投资建议，也不保证盈利。
